This is my custom Wave Function Collapse system (overlapping model) in Houdini — rule-driven, constraint-bound, and sometimes beautifully broken.
This WFC tool is still figuring out what it wants to become — and breaks most of the time.  
But I’m learning a lot, and sharing it in case someone out there sees something I don’t.
Work in progress. Always.
Built entirely in Houdini, the screenshots show clean structure on the surface.  
But beneath the nodes and logic, something about this system feels... alive.
Huge thanks to @mxgmn for the original algorithm 💡
And to @thecodingtrain for the amazing breakdown: https://youtu.be/5iSAvzU2WYY
